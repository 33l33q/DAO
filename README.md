# 프로젝트명 : **Java console application**
> DAO pattern을 기반으로 한 Java Console Application

----------

## 1.요건정리
> - 목적 :  예상 이용자 30명 내외를 대상으로한 긴급연락망 구축
> - 구조 : DAO pattern , java와 DBMS를 직접 연결하는 구조
> - 기능 : CRUD(입력, 수정, 삭제, 전체조회, 검색, like검색)
> - 제작 기간 : 14시간
-----------
